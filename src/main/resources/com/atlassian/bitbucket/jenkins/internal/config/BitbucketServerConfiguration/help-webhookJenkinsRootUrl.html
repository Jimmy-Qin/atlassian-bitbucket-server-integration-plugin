When the plugin registers a webhook against this Bitbucket Server instance, Bitbucket needs the Jenkins base URL that incoming webhook HTTP requests should use. The plugin appends the webhook path to this value.

Use this field if Bitbucket cannot reach Jenkins at the URL from global configuration—for example Jenkins is behind a reverse proxy and only a public URL works from Bitbucket’s network.

Leave empty to use the Jenkins URL from global configuration (Manage Jenkins → Configure System → Jenkins URL) when registering the webhook.
